# Resource Chains

Целевой дизайн ресурсных и энергетических цепочек для `Fulgur Fangs`.

Документ описывает желаемую экономику фракции.
Он не означает, что все перечисленное уже реализовано в `version-1.0`.

## Цели

- Разделить экономику на три понятных слоя:
  базовое строительство,
  электрические материалы,
  энергетическое топливо.
- Сохранить читаемость стоимостей зданий и рецептов.
- Не превращать электрическую ветку в набор из `8-10` обязательных ресурсов на каждое здание.
- Дать каждому ресурсу одну понятную роль.

## Правила Дизайна

- Базовые стройматериалы живут отдельно от электрических компонентов.
- Топливо для генерации живет отдельно от стройматериалов и отдельно от электрических компонентов.
- У раннего электрического здания должен быть `1` электрический маркер в стоимости.
- У среднего здания должен быть `1` собранный электрический компонент.
- У позднего здания может быть `1-2` продвинутых ресурса, но не весь склад сразу.
- Целевая стоимость здания:
  `2-3` базовых ресурса плюс `1` электрический ресурс своего тира.
- Целевой лимит:
  не более `3-4` типов ресурсов на одно здание.
- Жидкости допустимы как полноценный производственный ресурс:
  в Timberborn их можно таскать руками, отличается в основном хранение.

## 1. Базовый Слой

Это материалы до полноценной электроэры.
Они должны покрывать обычное строительство, раннюю механику и часть ранней электросети.

| Русское название | English name | Роль |
| --- | --- | --- |
| Металлолом | Scrap Metal | Базовое металлическое сырье |
| Бревна | Log | Базовая древесина и сырье для угля |
| Доски | Plank | Базовый строительный материал |
| Шестеренки | Gear | Стандартная механика и корпусные узлы |
| Металлические блоки | Metal Block | Базовый тяжелый строительный металл |

### Что входит в базовый слой

- обычные жилые и хозяйственные здания;
- ранние мастерские;
- простые генераторы;
- ранняя инфраструктура электросети.

### Что не надо делать в базовом слое

- Не добавлять сюда `Copper`, `ConductiveComponents`, `AccumulatorCell`.
- Не требовать редкоземельные материалы для простых зданий.
- Не смешивать топливо электростанций со стоимостью обычных построек.

## 2. Электрические Материалы

Это отдельный слой прогрессии поверх базовой экономики.
Он отвечает не за любое строительство, а именно за электронику, управление, накопление энергии и сложные машины.

| Русское название | English name | Статус | Роль |
| --- | --- | --- | --- |
| Медь | Copper | уже есть | Базовый электрометалл: проводка, контакты, катушки |
| Каучук | Rubber | уже есть | Изоляция, оболочки, защищенные соединения |
| Редкоземельный концентрат | Rare Earth Concentrate | уже есть | Поздний технологический усилитель |
| Проводящие компоненты | Conductive Components | уже есть | Собранный электрический промежуточный ресурс |
| Электродвигатель | Electric Motor | уже есть | Продвинутый узел для мощных машин и генераторов |
| Аккумуляторная ячейка | Accumulator Cell | уже есть | Специализированный поздний ресурс для накопителей и топовых зданий |

### Роли ресурсов

#### Медь / Copper

- Ранний электрический материал.
- Должна выплавляться в `Smelter`, рядом с `Metal Block`, а не собираться в `Metalsmith`.
- Используется в проводке, столбах, базовых конвертерах, простых электрических зданиях.
- Не должна быть поздним или редким ресурсом.

#### Каучук / Rubber

- Ресурс изоляции.
- Добавляет электрическим компонентам характер фракции.
- Не должен использоваться в каждом здании сети.

#### Редкоземельный Концентрат / Rare Earth Concentrate

- Поздний технологический материал.
- Хорошо работает как ограничитель для продвинутой электроники и аккумуляторов.
- Не должен быть обязательным ингредиентом для всей ранней электрической ветки.

#### Проводящие Компоненты / Conductive Components

- Главный собранный промежуточный ресурс электрической ветки.
- Именно он должен отделять "простую проводку" от "настоящей электроники".

#### Электродвигатель / Electric Motor

- Продвинутый компонент для тяжелых электрических машин.
- Нужен не в столбах и не в мелкой сети, а в мощных зданиях:
  генераторы,
  большие мастерские,
  конвертеры высокого тира.

#### Аккумуляторная Ячейка / Accumulator Cell

- Поздний специализированный ресурс.
- Основное применение:
  аккумуляторы,
  крупные накопители,
  несколько топовых энергозданий.
- Не должен стать обязательным строительным налогом на всю электронику.

## 3. Энергетический Слой

Это отдельная ветка, связанная именно с выработкой энергии.
Ее задача не строить здания, а кормить генерацию.

| Русское название | English name | Статус | Роль |
| --- | --- | --- | --- |
| Древесный уголь | Charcoal | уже есть | Базовое твердое топливо для тепловой генерации |
| Синтетическое топливо | Synthetic Fuel | уже есть | Позднее жидкое топливо для высокоэффективной станции |

### Почему именно так

- `Hydro Plant` не требует специального топлива и открывает электроэру без лишнего усложнения.
- `Coal Plant` дает игроку стабильную генерацию там, где гидроресурс слабый или нестабилен.
- `Synthetic Fuel` дает поздний скачок мощности без необходимости раздувать список электрических материалов.

## Необходимые Цепочки

### Базовая Металлургия

`Scrap Metal -> Metal Block`

Это ванильная база для обычного тяжелого строительства.

### Базовый Электрометалл

`Scrap Metal -> Copper`

Текущий ориентир:
`3 Scrap Metal -> 1 Copper` за `2.5 h`.

Производственное правило:

- `Copper` должна делаться в `Smelter`.
- `Metalsmith` остается зданием для деталей и собранных компонентов,
  а не для базовой выплавки металла.

### Базовая Электроника

Рекомендуемая целевая логика:

`Copper + Rubber -> Conductive Components`

Комментарий:

- `Conductive Components` должны быть первым "собранным" электрическим ресурсом.
- `Rare Earth Concentrate` лучше не делать обязательным ингредиентом этой ранней сборки.

### Продвинутый Электромеханический Узел

Рекомендуемая целевая логика:

`Copper + Metal Block + Rubber -> Electric Motor`

Комментарий:

- мотор связывает базовую металлургию с электрической веткой;
- мотор нужен мощным мастерским и генераторам;
- мотор не нужен мелкой инфраструктуре.

### Накопление Энергии

Рекомендуемая целевая логика:

`Conductive Components + Rare Earth Concentrate + Extract -> Accumulator Cell`

Комментарий:

- `Accumulator Cell` остается поздним ресурсом;
- `Extract` хорошо связывает badwater-экономику и электричество;
- `Rare Earth Concentrate` здесь уместен как ограничитель высокого тира.

### Твердое Топливо

Рекомендуемая целевая логика:

`Log -> Charcoal`

Комментарий:

- простой и понятный ранний переход от дерева к тепловой генерации;
- не требует дополнительной сложной химии.

### Позднее Жидкое Топливо

Рекомендуемая целевая логика:

`Charcoal + Extract -> Synthetic Fuel`

Комментарий:

- жидкий формат логичен для позднего топлива;
- `Extract` уже естественно связывает топливо с badwater-веткой;
- не нужен отдельный твердый ресурс вроде `Enriched Coal`, если жидкая ветка уже есть.

## Здания Генерации

### ГЭС / Hydro Plant

- Строится из базовых материалов.
- Не требует новых специализированных ресурсов.
- Работает от воды.
- Это вход в электроэру без лишнего давления на экономику.

### Угольная Электростанция / Coal Plant

- Строится в основном из базовых материалов.
- Может иметь небольшой электрический маркер, например `Copper` или `Conductive Components`.
- В работе использует `Charcoal`.

### Продвинутая Электростанция / Advanced Power Plant

- Строится из базовых материалов плюс поздние электрические компоненты.
- В работе использует `Synthetic Fuel`.
- Это поздний источник высокой стабильной генерации.

## Матрица Применения Ресурсов По Тирам

| Тир | Примеры зданий | Рекомендуемые ресурсы в стоимости |
| --- | --- | --- |
| База | обычные дома, склады, мастерские | `Log`, `Plank`, `Gear`, `Metal Block` |
| Ранний электрический | `Electric Pole`, `Grid Tester`, `Basic Dynamo Converter`, `Distributor v1` | база + `Copper` |
| Средний электрический | `Splitter`, `Transmission Tower`, `Advanced Dynamo Converter`, первые электрические мастерские | база + `Conductive Components` |
| Продвинутый электрический | `Coal Plant`, тяжелые мастерские, `Distributor v2` | база + `Conductive Components` + `Electric Motor` |
| Поздний энергетический | `Accumulator`, `Advanced Power Plant` | база + `Electric Motor` + `Accumulator Cell` |

## Ранний Вход В Электросеть

Чтобы у игрока была мотивация рано перейти в электричество,
первый сетевой тир должен запускаться без `Rubber`,
без `Rare Earth Concentrate`
и без собранных поздних компонентов.

### Правило Раннего Тира

- обязательные для запуска сети здания требуют только базу плюс `Copper`;
- `Copper` не требует электричества для производства;
- игрок должен суметь построить первую рабочую сеть,
  не поднимая еще вторую или третью технологическую ветку.

### Рекомендуемые Здания Раннего Тира

| Русское название | English name | Роль | Маркер тира |
| --- | --- | --- | --- |
| Базовый динамо-конвертер | Basic Dynamo Converter | Первый перевод механики в электричество | `Copper` |
| Электростолб | Electric Pole | Базовая локальная линия, `20 m`, потери `10 kW` | `Copper` |
| Базовый распределитель | Distributor v1 | Первая рабочая подстанция района | `Copper` |

### Почему Не Каучук На Базовом Распределителе

- `Distributor v1` не является optional QoL-апгрейдом.
- Это обязательная точка входа в локальную электросеть.
- Если повесить на него `Rubber` слишком рано,
  игрок сначала обязан поднять дополнительную производственную цепочку,
  прежде чем вообще увидит базовую работу электросети.

Текущая рекомендация:

- `Distributor v1` оставлять на `Copper`.
- `Rubber` начинать прятать внутрь `Conductive Components`,
  а не выносить отдельным строительным налогом в ранний сетевой тир.

## Сетевые Апгрейды И QoL-Здания

Эти здания не открывают электричество с нуля,
а улучшают удобство, дальность, масштаб и эффективность сети.
Поэтому им уже можно требовать собранные электрические компоненты.

| Русское название | English name | Почему выше тир | Рекомендуемый маркер |
| --- | --- | --- | --- |
| Разветвлитель | Splitter | Улучшает разводку сети и плотную застройку | `Conductive Components` |
| ЛЭП | Transmission Tower | Дает магистральную передачу на `200 m` при потерях `30 kW` | `Conductive Components` |
| Улучшенный динамо-конвертер | Advanced Dynamo Converter | Повышает масштаб и пропускную способность генерации | `Conductive Components` |
| Улучшенный распределитель | Distributor v2 | Улучшает покрытие и удобство снабжения | `Conductive Components` или `Conductive Components + Electric Motor` |

### Правило Сетевых Апгрейдов

- все здания, без которых первая рабочая сеть не стартует,
  требуют только `Copper`;
- все здания, которые улучшают дальность, удобство или масштаб сети,
  требуют `Conductive Components`;
- `Rubber` лучше использовать внутри рецепта `Conductive Components`,
  а не как отдельный массовый строительный ингредиент сетевых зданий.

## Что Нам Нужно

- Четкая трехслойная структура:
  стройматериалы,
  электроматериалы,
  топливо.
- `Copper` как ранний и массовый электрический материал.
- `Conductive Components` как основной промежуточный собранный ресурс.
- `Electric Motor` как отдельный продвинутый компонент.
- `Accumulator Cell` как ресурс узкого позднего применения.
- `Charcoal` как понятное раннее топливо.
- `Synthetic Fuel` как позднее жидкое топливо.

## Что Нам Не Нужно

- Не нужен отдельный ресурс `Cable` поверх `Copper`.
- Не нужен отдельный ресурс `Circuit Board` поверх `Conductive Components`.
- Не нужен отдельный ресурс `Magnet` поверх `Rare Earth Concentrate`.
- Не нужно делать `Rare Earth Concentrate` обязательным для всей ранней электроники.
- Не нужно требовать `Accumulator Cell` у обычных столбов, разветвителей и простой сети.
- Не нужно одновременно использовать в одном здании
  `Copper`,
  `Conductive Components`,
  `Electric Motor`,
  `Accumulator Cell`.

## Итоговая Целевая Схема

### Строительство

`Log -> Plank`

`Scrap Metal -> Metal Block`

### Электрика

`Scrap Metal -> Copper`

`Rubber Pine -> Rubber`

`Badwater -> Rare Earth Concentrate`

`Copper + Rubber -> Conductive Components`

`Copper + Metal Block + Rubber -> Electric Motor`

`Conductive Components + Rare Earth Concentrate + Extract -> Accumulator Cell`

### Энергетика

`Water flow -> Hydro Plant`

`Log -> Charcoal -> Coal Plant`

`Charcoal + Extract -> Synthetic Fuel -> Advanced Power Plant`

## Приоритет Реализации

1. Зафиксировать роли уже существующих ресурсов:
   `Copper`,
   `Rubber`,
   `Rare Earth Concentrate`,
   `Conductive Components`,
   `Accumulator Cell`.
2. Добавить `Coal Plant` и дать `Charcoal` первого полноценного потребителя.
3. Добавить позднюю электростанцию для `Synthetic Fuel`.
4. После этого уже точечно ребалансировать стоимости зданий по тиру.
